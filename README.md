# Notes App - Frontend

This is the frontend of the Notes application, built with **React**, **Redux Toolkit**, and **RTK Query**.  
It includes end-to-end tests using **Cypress**.

---

## Setup

1. **Clone the repository**

```bash
git clone https://github.com/Serg192/aionys-task-fe.git
cd aionys-task-fe
```

2. **Install dependencies**

```bash
npm install
```

3. **Set environment variables**

Set your api url (`REACT_APP_API_URL`) in `.env` file in the project root.

---

## Running the App

Start the development server:

```bash
npm start
```

---

## End-to-End Tests

Run Cypress E2E tests:

```bash
npm run test:e2e
```

while the web server is running.
